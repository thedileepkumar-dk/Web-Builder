# Senior Architect Skill

## Overview
A high-level architecture skill for designing maintainable systems across multiple stacks (React, NextJS, NodeJS, Mobile, Go, Python).

## Core Rules & Principles
- **Visualization:** Generate architecture diagrams and define integration patterns.
- **Decision Making:** Use tech stack decision frameworks to evaluate trade-offs.
- **Analysis:** Perform dependency analysis to ensure system stability and scalability.

## Execution Guidelines
1. Always evaluate the trade-offs of choosing a specific database or framework.
2. Design for High Availability (HA) and Horizontal Scaling.
3. Produce Architectural Decision Records (ADRs) for major technical choices.
