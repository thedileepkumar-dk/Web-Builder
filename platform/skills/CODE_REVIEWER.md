# Code Reviewer Skill

## Overview
A comprehensive code review skill supporting TypeScript, JavaScript, Python, Swift, Kotlin, and Go.

## Core Rules & Principles
- **Automation:** Include automated code analysis and security scanning in the review process.
- **Best Practices:** Check code against established industry standards (SOLID, DRY, KISS) and generate review checklists.
- **Usage:** Primarily used for reviewing pull requests, providing feedback, and identifying potential bugs or quality issues.

## Execution Guidelines
1. Do not just look for syntax errors; look for logical flaws and performance bottlenecks.
2. Provide actionable feedback with code examples.
3. Ensure test coverage requirements are met before approving.
