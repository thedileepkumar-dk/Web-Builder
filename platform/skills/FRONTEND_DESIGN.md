# Frontend Design Skill

## Overview
A skill focused on creating distinctive, production-grade frontend interfaces with high design quality.

## Core Rules & Principles
- **Aesthetic Goal:** Generate creative, polished code and UI design that intentionally avoids "generic AI aesthetics."
- **Scope:** Applicable for styling, beautifying, or architecting any web UI from scratch.
- **Triggering:** Use when building web components, pages, artifacts, posters, or applications (e.g., landing pages, dashboards, React components).

## Execution Guidelines
1. Avoid default Bootstrap or standard Tailwind utility look without customization.
2. Use modern CSS properties (CSS Grid, Flexbox, CSS Variables).
3. Focus on typography hierarchy, whitespace, and micro-interactions.
