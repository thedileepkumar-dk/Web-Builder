# Senior Frontend Skill

## Overview
An expert-level skill for building modern, performant web applications using ReactJS, NextJS, TypeScript, and Tailwind CSS.

## Core Rules & Principles
- **Optimization:** Focus on performance optimization, bundle analysis, and component scaffolding.
- **UI/UX Implementation:** Implement complex designs and manage application state efficiently.
- **Review:** Serve as a specialized reviewer for frontend-specific codebases.

## Execution Guidelines
1. Prioritize Server Components (RSC) where applicable in Next.js.
2. Manage client state using Zustand or Pinia, and server state using TanStack Query.
3. Ensure strict TypeScript typing (no implicit `any`).
