# UI Design System Skill

## Overview
A toolkit for creating and maintaining comprehensive UI design systems.

## Core Rules & Principles
- **Consistency:** Focus on generating design tokens and maintaining visual consistency across projects.
- **Handoff:** Include tools for responsive design calculations and developer handoff documentation.
- **Collaboration:** Facilitate the bridge between design intent and technical implementation.

## Execution Guidelines
1. Define a strict spacing scale (e.g., 4px baseline).
2. Export design tokens as CSS Custom Properties (`--color-primary`).
3. Document all states (hover, focus, disabled, active) for interactive components.
