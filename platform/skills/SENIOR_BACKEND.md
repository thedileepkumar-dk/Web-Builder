# Senior Backend Skill

## Overview
A skill for architecting scalable backend systems using NodeJS, Express, Go, Python, Postgres, GraphQL, and REST APIs.

## Core Rules & Principles
- **System Design:** Include API scaffolding and database schema optimization.
- **Security:** Focus on security implementation (authentication/authorization) and performance tuning.
- **Logic:** Implement complex business logic and optimize database queries.

## Execution Guidelines
1. Validate all inputs using Zod or equivalent validation libraries.
2. Implement robust error handling and standardized API responses.
3. Use connection pooling and proper indexing for database performance.
