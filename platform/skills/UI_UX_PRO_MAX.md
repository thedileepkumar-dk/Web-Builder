# UI/UX Pro Max Skill

## Overview
An advanced design intelligence skill featuring 50+ styles, 21 palettes, and support for 9 major frontend stacks.

- **Style Variety:** Master specific aesthetics like glassmorphism, claymorphism, minimalism, and bento grids.
- **Comprehensive Actions:** Cover the full lifecycle from planning and building to refactoring and optimizing UI/UX code.

## Execution Guidelines
1. Use distinct color palettes and spacing rules to create unique visual identities.
2. Implement complex CSS animations and Framer Motion transitions.
3. Ensure WCAG AAA compliance for high-contrast themes.
